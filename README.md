# デジキャンヘルパー

## Build with

- react
- typescript
- emotion

## Install

### 1. from chrome webstore

### 2. manually

1. clone source code

- `git clone git@github.com:rainy-me/digicam-ext.git`

2. install dependency

- `yarn`

3. build production

- `yarn build`

4. upload

- open [chrome://extensions/](chrome://extensions/)
- load unpacked in Developer mode
